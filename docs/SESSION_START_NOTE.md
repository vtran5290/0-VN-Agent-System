# Session start — nhắc mỗi khi bắt đầu thesis / khung chat mới

**Làm 1 việc trước khi đào sâu:**

1. Mở **`docs/RESEARCH_STATE.md`**.
2. Paste **toàn bộ nội dung** (hoặc @ file) vào **đầu** tin nhắn đầu tiên của conversation mới.

→ AI sẽ thấy: open questions, decisions đã đóng, current focus — tránh tunnel vision và miss big picture (e.g. câu hỏi 2012 extension bị chìm dưới MHC diagnostic).

---

**Nếu bắt đầu một thesis mới (ví dụ: persistence study, O'Neil overlay, backtest 2012):**

- Cập nhật RESEARCH_STATE.md: thêm open question hoặc current focus mới.
- Paste RESEARCH_STATE.md vào đầu chat.
- Thêm request theo **`docs/SESSION_PROMPT_TEMPLATES.md`** (có sẵn template 2012–2019 extension; điền "Câu hỏi cụ thể" rồi paste).

---

**Trước khi đóng session:** Update `RESEARCH_STATE.md` nếu có decision mới hoặc open question đã closed — nếu không, file sẽ stale sau 2–3 tuần.

*Institutional discipline: one anchor file, one paste, full map.*
